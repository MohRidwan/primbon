2. instal wordpress, extract di htdocs, rename sesuai nama project ex:wp, buat akun dll
3. bikin database nama sesuai project
4. edit file wp-config.php seuaikan dgn db yg kita buat di mysql
5. Seting permalink ramah SEO :
   5.1 dashboard,settings, permalinks, common settings, pilih post name, save changes

   6.1 dashboard, pages, add new,  kasih tittle dll, publish , publish
7. Buat Halaman Navigasi :
   7.1 dashboard, apperance, menus
   7.2 Menu structure, menu name : ex: master menu , create menu, menu setings, display location pilih primary
   7.3 Tambahkan halaman ke master menu :
       Untuk link custom : Add Menu items, custom links        
       Url : http://localhost/wp/
       Link Text : Home
       Add to menu

       save menu
